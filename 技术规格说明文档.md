## 1. PRD需求分析与解读

### 1.1 用户故事映射

#### US-001: 股票跟踪预测文件读取
**PRD原文**: 支持读取股票跟踪预测文件，自动解析关注股票列表

**技术解读**:
- 输入源: 用户提供的股票跟踪预测文件（CSV/JSON格式）
- 解析内容: 股票代码、股票名称、关注日期、跟踪状态等
- 数据验证: 校验股票代码格式、检查文件完整性

**验收标准**:
- [ ] 支持CSV和JSON两种文件格式
- [ ] 自动识别文件编码（UTF-8/GBK）
- [ ] 解析成功率 > 99%
- [ ] 错误信息清晰可读

**PRD原文**: 整合技术分析指标，识别价格趋势和买卖信号

**技术解读**:

**PRD原文**: 评估价值投资因子，判断股票内在价值与价格偏离度

| 指标 | 权重 | 评分标准 |
|-----|------|---------|

**估值模型**:
```python
# 简化DCF模型
def calculate_intrinsic_value(eps, growth_rate, discount_rate=0.1):
    """
    内在价值 = EPS * (1 + g) / (r - g)
    g: 增长率
    r: 折现率
    """
    if growth_rate >= discount_rate:
        return None
    return eps * (1 + growth_rate) / (discount_rate - growth_rate)

# 价格偏离度
def calculate_deviation(current_price, intrinsic_value):
    """
    偏离度 = (当前价格 - 内在价值) / 内在价值
    """
    return (current_price - intrinsic_value) / intrinsic_value
```

```python
# 基于关键词的情感分析
POSITIVE_KEYWORDS = ['利好', '增长', '突破', '创新高', '盈利', '收购']
NEGATIVE_KEYWORDS = ['利空', '下跌', '亏损', '风险', '调查', '处罚']

def analyze_sentiment(text: str) -> float:
    """
    返回情感得分: -1(极度负面) 到 1(极度正面)
    """
    positive_count = sum(1 for kw in POSITIVE_KEYWORDS if kw in text)
    negative_count = sum(1 for kw in NEGATIVE_KEYWORDS if kw in text)
    
    total = positive_count + negative_count
    if total == 0:
        return 0
    
    return (positive_count - negative_count) / total
```
