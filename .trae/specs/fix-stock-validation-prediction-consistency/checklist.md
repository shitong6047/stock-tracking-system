# 验证清单

## 股票代码验证功能
- [ ] 文件解析器能正确识别6位数字格式的股票代码
- [ ] 系统拒绝非数字、长度不正确的股票代码并给出明确错误信息
- [ ] 错误消息包含行号、无效代码内容和建议修正方案
- [ ] 支持批量处理时统计有效/无效股票数量
- [ ] 对已知的无效代码（如000620, 002210等）正确标记为无效

## 数据生成一致性
- [ ] 使用相同输入多次运行产生完全相同的输出结果
- [ ] 基于股票代码和日期的哈希种子正常工作
- [ ] 所有随机数替换为确定性算法，无残留的真正随机调用
- [ ] 数据生成的性能开销在可接受范围内（<10%）
- [ ] 生成的数据在合理范围内（价格、涨跌幅等符合实际市场规律）

## 预测模型稳定性
- [ ] 相同股票+相同历史数据产生相同的预测方向
- [ ] 概率值变化范围在±0.01容差内（浮点精度）
- [ ] 模型训练使用固定随机种子，可重复训练过程
- [ ] 简单规则预测模式完全确定性，无任何随机因素
- [ ] 预测缓存机制正常工作，避免不必要的重复计算

## 错误处理增强
- [ ] 主程序在遇到无效股票时不崩溃，继续处理其他股票
- [ ] 执行报告包含详细的成功/失败/跳过统计
- [ ] 日志记录足够详细，便于问题定位
- [ ] 数据获取失败时有合理的降级方案
- [ ] 用户界面反馈清晰，区分警告和错误
