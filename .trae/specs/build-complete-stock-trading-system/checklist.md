# 验证清单

## Phase 1: 核心架构与数据层

### Task 1: 主程序架构重构
- [ ] 系统支持三种运行模式：screen（市场扫描）、analyze（指定分析）、both（两者兼做）
- [ ] 命令行参数解析完整，支持所有配置选项（--mode, --file, --output, --scope, --top-n等）
- [ ] 配置管理模块正常工作，支持命令行 > 环境变量 > config.json 的优先级
- [ ] 统一日志系统运行正常，可输出到控制台和文件
- [ ] 错误处理机制完善，异常情况不会导致程序崩溃

### Task 2: 数据获取能力增强
- [ ] 可以获取A股全市场股票列表（沪深主板+创业板+科创板，3000+只）

- [ ] 历史K线数据获取正常（至少60个交易日）
