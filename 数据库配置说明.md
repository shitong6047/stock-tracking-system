# Supabase数据库配置说明

## 数据库配置

系统支持两种数据库连接方式：

### 1. Supabase连接（推荐）

Supabase提供PostgreSQL数据库服务，配置简单，适合生产环境。

#### 配置步骤：

1. 访问 [Supabase](https://supabase.com/) 注册账号
2. 创建新项目
3. 在项目设置中获取API密钥
4. 将以下配置添加到 `.env` 文件：

```bash
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_KEY=your-anon-key
```

5. 安装Supabase依赖：
```bash
pip install supabase
```

### 2. PostgreSQL直接连接

如果已有PostgreSQL数据库，可以直接连接。

#### 配置步骤：

1. 确保数据库已创建
2. 将以下配置添加到 `.env` 文件：

```bash
DATABASE_URL=postgresql://username:password@host:port/database_name
```

例如：
```bash
DATABASE_URL=postgresql://postgres:your-password@db.veztbosecvculqruhkqp.supabase.co:5432/postgres
```

3. 安装psycopg2依赖（已包含在requirements.txt中）：
```bash
pip install psycopg2-binary
```

## 数据库表结构

系统会自动创建以下表：

### stock_pool（股票池表）
- id: 主键
- create_time: 创建时间
- update_time: 更新时间
- stocks: 股票数据（JSONB格式）

### tracking_log（跟踪日志表）
- id: 主键
- created_at: 创建时间
- stock_code: 股票代码
- stock_name: 股票名称
- latest_price: 最新价格
- price_change_pct: 涨跌幅
- volume: 成交量
- tracking_note: 跟踪备注
- alert_signal: 异常信号

### predictions（预测结果表）
- id: 主键
- created_at: 创建时间
- code: 股票代码
- name: 股票名称
- direction: 预测方向（上涨/下跌/震荡）
- probability: 预测概率
- confidence: 置信度
- total_score: 总得分
- key_signals: 关键信号（JSONB格式）
- risk_warning: 风险提示

### global_news（国际新闻表）
- id: 主键
- created_at: 创建时间
- title: 新闻标题
- summary: 新闻摘要
- pub_time: 发布时间
- source: 新闻来源
- url: 新闻链接
- impact: 影响方向（乐观/悲观）
- impact_score: 影响分数

## 故障排查

### 1. Supabase连接失败

**错误信息**: `name 'create_client' is not defined`

**解决方案**: 安装supabase库
```bash
pip install supabase
```

### 2. PostgreSQL连接失败

**错误信息**: `password authentication failed`

**解决方案**: 
- 检查DATABASE_URL中的密码是否正确
- 确保数据库用户有连接权限
- 检查防火墙设置

### 3. 自动降级到文件存储

如果数据库连接失败，系统会自动降级到文件存储模式：
- 股票池: `data/stock_pool.json`
- 跟踪日志: `data/stock_tracking_log.csv`
- 预测结果: `data/prediction_*.json`
- 国际新闻: `data/global_news.json`

## 安全建议

1. **不要提交敏感信息**: `.env` 文件已包含在 `.gitignore` 中
2. **使用环境变量**: 生产环境使用环境变量存储数据库连接信息
3. **定期备份**: 定期备份数据库数据
4. **限制权限**: 数据库用户只授予必要权限

## 监控和维护

### 查看数据库状态
```python
from src.database import SupabaseStorage

storage = SupabaseStorage()
print(f"连接状态: {storage.is_connected}")
```

### 查看跟踪日志
```python
logs = storage.get_tracking_logs(limit=10)
for log in logs:
    print(log)
```

### 查看股票池
```python
pool = storage.get_stock_pool()
if pool:
    print(f"股票池包含 {len(pool)} 只股票")
```
